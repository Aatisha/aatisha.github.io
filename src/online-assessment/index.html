Problem Statement

In the present educational institutions for programming labs, teachers provide problem statements to students, and students submit their solutions per the problem statement provided. Grading of these solutions is done manually. It is a time-consuming task. To determine the solution's correctness, they are required to run multiple test cases. Also, the faculty has to ensure that each computer in the lab has an editor and compiler to run the code.

Overview

The proposed web application will provide guidelines for a system interface for teachers to create new programming assignments and assign them to a particular class. Also, it helps teachers with the grading of programming assignments. It will provide a detailed report of the class for each programming assignment. It will also reduce the overhead of installing the software required to edit and run the code. The system provides an easy interface for students to edit and compile programs. It will also give a detailed analysis of evaluated assignments. An additional feature of the chatbot is also available.
